## Doing exercises
#### 进入 exercises/cos_arceos/exercises目录下找到对应的题目
#### 第一种方式：
1. 将代码clone到本地并执行 cargo install --path .
2. 上个步骤会将arceos安装到本地, 输入命令 arceos run xx可以xx对应的课程作业;例如 arceos run a0
3. 提交代码到github

#### 第二种方式：
1. 进入exercises目录下直接运行 ./varify xx 例如: ./varify a0 进行作业验证
2. 代码提交
